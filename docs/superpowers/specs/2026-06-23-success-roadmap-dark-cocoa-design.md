# 성공 로드맵 다크 코코아 디자인

## 목표

`성공으로 가는 6단계 로드맵`의 밝은 한지·아이보리 인상을 줄이고, 로고의 흑갈색과
구리빛을 반영한 묵직하고 고급스러운 섹션으로 재구성한다.

## 색상 배치

- 섹션 배경은 `#2a1d16`에서 `#3b291e`로 이어지는 다크 코코아 그라데이션을 사용한다.
- 제목은 따뜻한 아이보리 `#f0dfc0`, 설명은 탁한 샌드 `#c8b69a`를 사용한다.
- 단계 카드는 구운 베이지 `#d3b98e`를 중심으로 구성한다.
- 카드 제목은 먹갈색 `#291911`, 본문은 대비를 확보한 코코아 `#594334`를 사용한다.
- 번호 배지는 적갈색 `#713e25`, 상단 포인트와 구분선은 구리색 `#b56e32`를 사용한다.
- 테두리는 앤티크 골드 `#a9824c`를 낮은 투명도로 사용한다.

## 배경과 카드

- 기존 한지 이미지는 질감으로만 남기고 어두운 코코아 베일을 강하게 겹친다.
- 카드 레이아웃, 여섯 단계 데이터, 이미지 비율은 변경하지 않는다.
- 카드 이미지는 밝기와 채도를 살짝 낮추고 갈색 하단 베일을 추가한다.
- 카드 그림자는 검은색 대신 짙은 코코아 계열로 깊게 표현한다.
- 호버 동작은 기존 상승 동작을 유지하되 그림자를 더 무겁게 만든다.

## 타이틀과 마무리 패널

- 영문 라벨과 양옆 선은 구리색으로 통일한다.
- 제목과 설명은 밝은 웜톤으로 두어 어두운 배경에서 충분한 대비를 확보한다.
- 마지막 CTA는 카드와 반대로 짙은 초콜릿 패널로 구성한다.
- CTA 제목은 아이보리, 설명은 샌드, 상단 선은 구리색을 사용한다.

## 유지 범위

- 문구, 단계 데이터, 이미지, 반응형 그리드, 등장 순서, 모션 감소 대응은 유지한다.
- 다른 섹션과 전역 CSS는 변경하지 않는다.

## 검증

- 새 색상 토큰과 기존 구조 보존 여부를 Node 계약 테스트로 확인한다.
- 기존 공통 섹션 계약은 밝은 공유 팔레트를 앞의 두 섹션에만 적용하고, 로드맵에는 장식 제거 규칙만 유지한다.
- 대상 컴포넌트와 계약 테스트에 ESLint를 실행한다.
- 전체 Next.js 프로덕션 빌드를 실행한다.
- 로컬 페이지에서 데스크톱과 모바일의 대비와 카드 가독성을 확인한다.
