# 리포트 톤 헤더·하단 가맹문의 디자인

## 목표

현재 짙은 브라운 중심의 헤더와 하단 고정 가맹문의 UI를 “전국 오픈 리포트” 팝업과 같은 밝은 편집형 톤으로 통일한다. 페이지 위에 무거운 검은 띠가 두 개 떠 있는 인상을 줄이고, 아이보리 종이·적갈색 포인트·잉크색 타이포가 이어지는 하나의 브랜드 시스템으로 보이게 한다.

## 선택한 방향

- 기본 배경은 팝업과 동일한 밝은 아이보리 `#f6efe3` 계열을 사용한다.
- 주요 구분선과 CTA는 적갈색 `#8f3528`, 보조 정보와 포커스는 금색 `#c9a24d`를 사용한다.
- 장식은 얇은 상단 룰, 미세한 점 패턴, 작은 영문 라벨로 제한한다.
- 둥근 카드와 그림자는 유지하되 과도한 글로우, 진한 그라데이션, 검은 배경은 제거한다.

## 헤더

- 고정 헤더는 밝은 종이 패널과 얇은 하단 보더를 사용한다.
- 최상단에 가는 적갈색 브랜드 룰을 둔다.
- 로고, 내비게이션, 창업문의 CTA의 세 영역을 명확히 분리한다.
- 내비게이션은 잉크색 기본 텍스트와 적갈색 호버/포커스 상태를 사용한다.
- 창업문의 CTA는 적갈색 배경과 아이보리 텍스트로 강조한다.
- 모바일 메뉴는 같은 아이보리 패널, 번호가 붙은 메뉴 목록, 적갈색 문의 CTA로 구성한다.
- 기존 스크롤 이동과 모바일 메뉴 열기/닫기 동작은 유지한다.

## 하단 빠른 가맹문의

- 데스크톱은 한 줄짜리 밝은 리포트 데스크 형태로 유지한다.
- 좌측에는 `FRANCHISE DESK` 라벨, 빠른 가맹문의 제목, 전화번호를 계층적으로 표시한다.
- 입력 필드는 따뜻한 흰색 배경과 차분한 보더를 사용하고 포커스 시 적갈색/금색 링을 제공한다.
- 개인정보 동의는 보조 정보로 낮추되 체크 영역과 라벨은 충분한 클릭 범위를 유지한다.
- 제출 버튼은 적갈색 배경으로 만들고 비활성 상태는 명도와 투명도로 구분한다.
- 모바일 확장 패널은 같은 종이 톤의 바텀 시트로, 축소 상태는 작은 리포트 카드로 표시한다.
- 모바일 접기 버튼은 상단 핸들과 텍스트 버튼을 결합해 현재 기능을 명확히 유지한다.

## 기능과 데이터

- 히어로를 지난 뒤 노출되고 문의 섹션 또는 푸터가 보일 때 숨는 조건을 유지한다.
- 모바일 자동 확장, 사용자 접기, 다시 열기 동작을 유지한다.
- 성함, 연락처, 희망지역, 개인정보 동의, 허니팟 필드와 제출 API payload를 변경하지 않는다.
- 유효성 검사, 성공/실패 알림, 제출 중 상태를 유지한다.

## 반응형과 접근성

- 320px 모바일에서도 입력 필드와 CTA가 화면 폭 안에 들어온다.
- 태블릿과 데스크톱에서는 문의 바가 두 줄로 깨지지 않도록 가변 폭을 사용한다.
- 전화번호 링크, 메뉴 버튼, 제출 버튼의 접근 가능한 이름을 유지한다.
- 포커스 상태와 텍스트 대비가 밝은 배경에서도 명확해야 한다.
- 안전 영역 패딩을 유지해 iOS 하단 UI와 겹치지 않게 한다.

## 성공 기준

- 헤더와 하단 문의 UI에서 짙은 브라운 전체 배경이 제거된다.
- 두 컴포넌트가 오픈 리포트 팝업과 동일한 아이보리·적갈색·금색 팔레트로 보인다.
- 데스크톱과 모바일의 기존 동작 및 문의 제출 데이터가 유지된다.
- 모바일 메뉴와 하단 문의 패널이 작은 화면에서 잘리지 않는다.
- 정적 UI 계약, ESLint, 프로덕션 빌드가 통과한다.
