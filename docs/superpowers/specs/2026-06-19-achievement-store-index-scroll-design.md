# 전국 오픈 리포트 매장 목록 스크롤 설계

## 목표

가맹점 수가 늘어나도 `전국 오픈 리포트` 모달이 화면 높이를 넘지 않도록 매장 목록만
내부 스크롤 영역으로 만든다.

## 레이아웃

- 리포트 헤더와 범례는 기존 위치에 고정한다.
- `STORE INDEX` 제목과 설명은 목록 위에 고정한다.
- 실제 매장 행을 담는 `<ol>`에만 최대 높이와 `overflow-y-auto`를 적용한다.
- 하단 `FRANCHISE INQUIRY` 안내는 목록 아래에 고정한다.
- 모바일과 데스크톱 모두 화면 높이에 비례하는 목록 최대 높이를 사용한다.
- 기존 매장 정렬, 상태 표시, 로딩 및 빈 목록 처리는 변경하지 않는다.

## 스크롤 동작

- 목록이 최대 높이보다 짧으면 스크롤바 없이 기존 높이만 사용한다.
- 목록이 최대 높이를 넘으면 목록 내부에서만 세로 스크롤한다.
- 스크롤이 모달 바깥 페이지로 연쇄되지 않도록 overscroll을 제한한다.

## 검증

- 소스 계약 테스트로 `<ol>`에 목록 전용 최대 높이, 세로 스크롤, overscroll 제한이
  적용되었는지 확인한다.
- `STORE INDEX` 헤더와 `FRANCHISE INQUIRY`가 스크롤 컨테이너 밖에 있는지 확인한다.
- ESLint와 프로덕션 빌드를 실행한다.
