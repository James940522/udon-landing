# 고객 리뷰 섹션 웜 페이퍼 무대 설계

## 목표

`고객들이 다시 찾는 볶음우동` 리뷰 섹션을 최근 정리한 크림, 적갈색, 브라운 중심의
톤앤매너에 맞춘다. 실제 배달앱 화면과 고객 리뷰 이미지가 증거물로 또렷하게 보이는 구조는
유지한다.

## 확정 디자인

### 섹션 배경과 헤더

- 섹션 전체의 검은 배경을 밝은 크림·한지 계열로 변경한다.
- 기존 오렌지 글로우, 금색 격자, 세로 장식선과 기하학 장식은 제거한다.
- 배경에는 낮은 대비의 종이 그라데이션과 작은 점무늬만 남긴다.
- 영문 라벨은 적갈색, 메인 제목은 짙은 브라운으로 변경한다.
- `다시 찾는 볶음우동`은 적갈색으로 강조해 제목의 정보 위계를 만든다.
- 우측 설명의 소제목은 적갈색, 본문은 중간 브라운을 사용한다.

### 휴대폰 무대

- 두 휴대폰을 감싸는 하나의 짙은 브라운 패널을 추가한다.
- 패널은 둥근 모서리, 얇은 적갈색·골드 계열 테두리와 브라운 그림자를 사용한다.
- 패널 내부에는 낮은 대비의 적갈색·골드 방사형 그라데이션을 사용해 휴대폰 화면을 강조한다.
- 휴대폰 이미지 크기, 위치, 모바일 2열 구조와 데스크톱 여백은 유지한다.
- 평점 라벨은 짙은 갈색 대신 적갈색 바탕과 아이보리 텍스트로 변경한다.

### 리뷰 마퀴

- 하단 리뷰 마퀴의 밝은 종이 배경과 모바일 배치 순서는 유지한다.
- 리뷰 이미지 카드의 금색 테두리는 눌린 베이지 테두리로 조정한다.
- 카드 그림자는 기존보다 중립적인 브라운으로 정리한다.
- 마퀴 애니메이션 속도와 이미지 데이터는 변경하지 않는다.

## 비변경 범위

- 배달앱 이미지, 리뷰 이미지와 대체 텍스트는 변경하지 않는다.
- `PhonePreview`의 크기와 반응형 최대 너비는 변경하지 않는다.
- 리뷰 마퀴가 모바일에서 휴대폰 아래에 위치하는 구조는 유지한다.
- 연락처 섹션과 가맹절차 섹션은 변경하지 않는다.
- 외부 패키지와 새 이미지 에셋은 추가하지 않는다.

## 검증

- 리뷰 섹션 계약 테스트에 밝은 종이 배경, 짙은 휴대폰 무대, 적갈색 강조와 베이지 리뷰 카드
  토큰을 추가한다.
- 기존 검은 섹션 배경과 강한 장식용 그라데이션이 제거됐는지 확인한다.
- 기존 모바일 마퀴 배치 계약을 유지한다.
- 계약 테스트를 RED에서 GREEN으로 확인하고, 집중 ESLint, 전체 Node 테스트와 프로덕션
  빌드를 실행한다.
