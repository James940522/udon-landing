# 밝고 확장된 ABOUT US 섹션 설계

## 목표

`ABOUT US` 섹션의 음식 사진은 유지하면서 현재의 검고 압축된 인상을 걷어내고, 이후
순차적으로 수정할 섹션들의 밝은 리포트 톤 기준을 만든다.

## 시각 방향

- 기존 검정 오버레이를 제거하고 아이보리·크림색 베일을 적용한다.
- 사진은 밝기와 채도를 부드럽게 보정해 배경의 온도감만 남긴다.
- 섹션 하단에는 옅은 적갈색 그라데이션을 사용해 다음 섹션과 깊이 차이를 만든다.
- 제목과 본문은 흰색 대신 잉크색·브라운 계열로 전환한다.
- 포인트 색상은 기존 리포트 톤과 동일한 적갈색과 금색을 사용한다.

## 높이와 레이아웃

- 섹션은 `min-h-[clamp(48rem,92vh,64rem)]`으로 확장한다.
- 세로 패딩은 모바일 `py-24`, 태블릿 `md:py-32`, 데스크톱 `lg:py-40`을 사용한다.
- 콘텐츠 폭을 `max-w-4xl`에서 `max-w-6xl`로 넓혀 카드 여백을 확보한다.
- 제목과 카드 사이 간격도 늘려 섹션이 숨 쉴 수 있게 한다.

## 카드

- 반투명 검정 카드 대신 반투명 아이보리 종이 카드를 사용한다.
- 따뜻한 금색 테두리, 옅은 브라운 그림자, 작은 블러를 적용한다.
- 카드 제목은 잉크색, 영문 라벨은 적갈색, 설명은 중간 브라운으로 표시한다.
- 기존 세 가지 브랜드 가치와 애니메이션 동작은 유지한다.

## 비변경 범위

- 배경 이미지 파일과 텍스트 내용은 변경하지 않는다.
- `useInView` 및 Framer Motion 애니메이션은 유지한다.
- 다른 섹션은 이번 변경에 포함하지 않는다.

## 검증

- 소스 계약 테스트에서 새 높이, 밝은 베일, 밝은 카드 및 텍스트 색상을 확인한다.
- 기존 검정 오버레이와 어두운 카드 토큰이 제거되었는지 확인한다.
- ESLint와 프로덕션 빌드를 실행한다.
- 개발 서버는 실행하지 않는다.
