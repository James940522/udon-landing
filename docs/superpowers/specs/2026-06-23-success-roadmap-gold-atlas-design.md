# 성공 로드맵 골드 아틀라스 배경 장식 디자인

## 목표

다크 코코아 색상과 구운 베이지 카드는 유지하면서, 비어 보이는 배경에 로드맵의 흐름과
브랜드의 금장 그릇 이미지를 연상시키는 화려한 골드 아틀라스 장식을 추가한다.

## 장식 구성

- 섹션 안쪽에 두 겹의 앤티크 골드 프레임과 장식 모서리를 배치한다.
- 제목 뒤에 동심원과 방사형 선으로 구성한 반원형 골드 광륜을 배치한다.
- 카드 사이 여백을 통과하는 구리색 이중 점선 경로와 금장 이중 원형 메달을 배치한다.
- `01–06` 대형 숫자는 카드 아래가 아니라 섹션 가장자리와 카드 사이로 일부가 드러나게 배치한다.
- 좌상단에는 구름을 추상화한 선화, 우하단에는 물결을 추상화한 선화를 크게 배치한다.
- 모든 장식은 `pointer-events-none`이며 콘텐츠보다 낮은 레이어에 둔다.

## 밀도와 색상

- 경로는 `#d29a52`를 32–38% 투명도로 사용하고 바깥쪽에 흐린 광택선을 겹친다.
- 대형 숫자는 `#f0dfc0`를 10–12% 투명도로 사용한다.
- 모서리 선화는 앤티크 골드 `#c7a061`를 20–24% 투명도로 사용한다.
- 이중 프레임은 바깥쪽 28%, 안쪽 14% 투명도로 차등 적용한다.
- 카드와 제목 뒤에는 장식이 직접 겹쳐 읽기 어려워지지 않도록 충분한 여백과 낮은 투명도를 유지한다.

## 반응형 처리

- 데스크톱과 태블릿에서는 이중 프레임, 광륜, 경로, 메달, 대형 숫자를 모두 표시한다.
- 모바일에서는 복잡한 전체 경로와 대형 숫자를 숨긴다.
- 모바일에는 이중 프레임, 축소된 광륜, 모서리 선화를 유지한다.

## 유지 범위

- 기존 다크 코코아 팔레트, 카드 디자인, 문구, 이미지, 애니메이션, 반응형 그리드는 변경하지 않는다.
- 장식은 정적 SVG와 텍스트 레이어만 사용하며 추가 이미지 자산은 만들지 않는다.

## 검증

- 장식 레이어의 식별자, 투명도, 모바일 숨김 규칙을 계약 테스트로 확인한다.
- 카드 6개, 모바일 한 열, 데스크톱 세 열, 모션 감소 대응이 유지되는지 확인한다.
- ESLint, Prettier, Next.js 프로덕션 빌드와 브라우저 렌더링을 검증한다.
