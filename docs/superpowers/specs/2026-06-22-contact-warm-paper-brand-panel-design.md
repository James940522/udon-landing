# 창업 문의 섹션 웜 페이퍼 브랜드 패널 설계

## 목표

`성공 창업, 오늘은 볶음우동입니다.` 창업 문의 섹션을 최근 정리한 크림, 적갈색, 브라운
중심의 톤앤매너에 맞춘다. 밝아진 고객 리뷰 섹션에서 자연스럽게 이어지고, 아래의 어두운
푸터로 부드럽게 전환되도록 한다.

문의 메시지와 폼의 명확한 대비는 유지하면서 기존 제출 동작과 입력 구조는 변경하지 않는다.

## 확정 디자인

### 섹션 배경과 연결

- 섹션 전체의 검은 배경을 밝은 크림·한지 계열로 변경한다.
- 기존 금빛 대각선, 격자와 회전 사각형 장식은 제거한다.
- 배경에는 낮은 대비의 종이 그라데이션과 작은 점무늬를 사용한다.
- 섹션 하단은 짙은 브라운 그라데이션으로 마감해 어두운 푸터와 연결한다.

### 브랜드 메시지 패널

- 왼쪽 메시지 카드는 금빛 그라데이션 대신 짙은 초콜릿 브라운 패널로 변경한다.
- 패널 상단에는 적갈색 라인을 추가해 최근 카드 디자인과 공통 언어를 만든다.
- `Today Udon` 라벨은 적갈색 바탕과 아이보리 텍스트를 사용한다.
- `성공 창업,`은 아이보리, `오늘은 볶음우동`은 따뜻한 골드로 강조한다.
- 설명은 밝은 베이지로 조정해 패널 안에서 충분한 대비를 확보한다.
- 전화번호 CTA는 아이보리 바탕, 적갈색 아이콘·텍스트와 브라운 그림자를 사용한다.

### 문의 폼

- 오른쪽 폼 카드는 또렷한 아이보리 바탕과 눌린 베이지 테두리로 변경한다.
- 폼 상단에 얇은 적갈색 라인을 추가해 메시지 패널과 한 쌍으로 보이게 한다.
- 입력창과 선택 버튼의 배경은 밝은 크림, 테두리는 눌린 베이지를 사용한다.
- 선택된 옵션과 제출 버튼은 브랜드 적갈색을 유지한다.
- 개인정보 안내 영역은 연한 베이지 바탕과 베이지 테두리로 정리한다.
- 라벨, 필수 표시와 안내 문구는 짙은 브라운·적갈색 체계를 유지한다.

## 반응형과 동작

- 모바일에서는 현재처럼 메시지 패널 다음에 문의 폼이 배치된다.
- 데스크톱의 2열 비율과 최대 너비는 유지한다.
- 입력 상태, 유효성 검사, 개인정보 동의, API 요청, 성공·실패 안내는 변경하지 않는다.
- SMS 링크, 전화번호, 폼 필드명과 제출 데이터는 변경하지 않는다.

## 비변경 범위

- 고객 리뷰 섹션과 푸터는 수정하지 않는다.
- 문구, 폼 데이터, 이벤트 핸들러와 API 호출은 변경하지 않는다.
- 외부 패키지와 새 이미지 에셋은 추가하지 않는다.

## 검증

- 전용 소스 계약 테스트를 추가해 밝은 종이 배경, 짙은 메시지 패널, 아이보리 폼,
  적갈색 상단 라인과 푸터 연결 그라데이션을 확인한다.
- 기존 검은 섹션 배경과 금빛 메시지 카드 그라데이션이 제거됐는지 확인한다.
- 폼 필드, SMS 링크, API 경로와 제출 상태 문구가 유지되는지 확인한다.
- 계약 테스트를 RED에서 GREEN으로 확인하고 집중 ESLint, 전체 Node 테스트와 프로덕션
  빌드를 실행한다.
