# 헤더 높이·브랜드 가독성 조정 설계

## 목표

사이트 상단 헤더의 세로 공간은 줄이면서 로고와 내비게이션 텍스트의 존재감은 키운다.

## 변경

- 헤더 콘텐츠 높이를 모바일 `h-16`에서 `h-14`로 줄인다.
- 데스크톱 높이를 `md:h-[72px]`에서 `md:h-16`으로 줄인다.
- 로고 높이를 모바일 `h-10`에서 `h-11`, 데스크톱 `md:h-12`에서 `md:h-14`로 키운다.
- 데스크톱 메뉴와 창업문의 텍스트를 `text-sm`에서 `text-base`로 키운다.
- 메뉴 버튼의 세로 패딩은 `py-2`에서 `py-1.5`, 창업문의는 `py-2.5`에서 `py-2`로 줄인다.
- 데스크톱 보조 라벨은 각각 1px 키운다.
- 모바일 메뉴 버튼과 사이드 시트의 크기는 변경하지 않는다.

## 검증

- 소스 계약 테스트로 새 높이·로고·텍스트·패딩 값을 확인한다.
- 기존 높이와 텍스트 크기 조합이 제거되었는지 확인한다.
- ESLint와 프로덕션 빌드를 실행한다.
- 사용자의 요청에 따라 개발 서버는 실행하지 않는다.
