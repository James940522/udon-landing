# 창업 혜택 섹션 한지 배경 디자인

## 목적

“초기 부담은 낮추고 / 필요한 것만 준비” 섹션을 제공된 밝은 한지 이미지 중심으로 재구성한다. 기존 정보 구조와 `無` 도장 애니메이션은 유지하면서 어두운 장식을 걷어내고, 종이 질감과 조화되는 따뜻하고 읽기 쉬운 컬러 체계로 변경한다.

## 배경 자산

- 제공된 1672×941 PNG 이미지를 WebP로 변환한다.
- 변환된 파일은 `public/new-asset/background/startup-benefit-hanji.webp`에 저장한다.
- 섹션 전체 배경에 `cover`, 중앙 정렬로 적용한다.
- 화면 폭이 넓거나 길어져 이미지가 일부 잘리더라도 종이 질감이 자연스럽게 이어지도록 밝은 크림색 배경을 폴백으로 둔다.

## 제거 대상

현재 섹션의 다음 장식은 모두 제거한다.

- 검정·갈색 방사형 및 선형 그라데이션 배경
- 양옆 세로 영문 마키 문구
- 상단 금색 가로선
- 좌우 금색 세로선
- 배경 장식용 데이터와 마키 애니메이션 참조

콘텐츠인 제목, 설명, 표, 하단 안내 문구는 유지한다.

## 컬러 체계

밝은 한지 위에서 정보 위계가 선명하도록 다음 팔레트를 사용한다.

- 섹션 기본 글자: 먹색 `#26140e`
- 영문 포인트와 강조색: 옻칠 적갈색 `#8f3528`
- 설명 및 보조 문구: 부드러운 갈색 `#725744`
- 표 외곽 패널: 반투명 크림 `rgba(255, 248, 235, 0.86)`
- 표 헤더·분류·금액 셀: 짙은 갈색 `#3b2115`
- 표 내용 셀: 밝은 크림 `#fff8eb`
- 표 테두리: 베이지 골드 `#c9a24d`의 낮은 투명도

표의 구조와 열 비율은 유지한다. 어두운 셀의 텍스트는 크림색, 밝은 셀의 텍스트는 먹색으로 설정한다.

## 레이아웃

- 기존 최대 너비와 반응형 열 구성을 유지한다.
- 제목과 설명의 배치는 유지하되 어두운 배경용 그림자를 제거하거나 종이 위에서 자연스러운 약한 그림자로 바꾼다.
- 표 주변에는 종이와 표를 구분하는 반투명 크림 패널, 얇은 테두리, 부드러운 그림자만 사용한다.
- 별도의 배경 장식 요소는 추가하지 않는다.

## 애니메이션

- 제목 블록은 아래에서 위로 올라오며 나타난다.
- 설명은 제목보다 약간 늦게 나타난다.
- 표 컨테이너가 나타난 뒤 각 행이 위에서부터 짧은 간격으로 순차 등장한다.
- 하단 안내 문구도 표 이후 부드럽게 나타난다.
- 애니메이션은 이동 거리가 짧고 투명도 변화 중심으로 구성해 정보 읽기를 방해하지 않는다.
- 운영체제의 모션 감소 설정에서는 이동 애니메이션을 최소화한다.

## `無` 도장 보존

`StampMark` 컴포넌트의 모양, 색상, 진입 조건, 타이밍, 스케일 및 회전 애니메이션은 변경하지 않는다. 표 행 애니메이션은 도장 컴포넌트 바깥에만 적용해 기존 도장 동작에 영향을 주지 않는다.

## 검증

- 제공된 PNG와 동일한 비율의 WebP 파일이 생성되는지 확인한다.
- 섹션 소스에서 기존 그라데이션, 금색 외곽선, 세로 마키 데이터가 제거됐는지 확인한다.
- 새 WebP 경로가 섹션 배경으로 사용되는지 확인한다.
- `StampMark` 구현이 변경되지 않았는지 확인한다.
- 모바일과 데스크톱에서 제목, 설명, 표의 대비와 여백을 확인한다.
- 프로덕션 빌드를 실행해 이미지 경로, TypeScript, CSS 오류가 없는지 확인한다.

## 성공 기준

- 섹션 전체에 밝은 한지 질감이 자연스럽게 표시된다.
- 기존의 어두운 배경 장식은 보이지 않는다.
- 제목·설명·표는 기존 정보 구조를 유지하면서 한지 배경에 맞는 색으로 읽기 쉽다.
- 표 행과 콘텐츠의 애니메이션은 자연스럽고, `無` 도장 애니메이션은 이전과 동일하게 작동한다.
