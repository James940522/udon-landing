# 헤더·푸터 컬러 농도 조정 설계

## 목표

현재의 크림·브라운 브랜드 팔레트를 유지하면서 헤더와 푸터 배경만 한 단계 진하게 조정해 화면의 시작과 끝을 더 또렷하게 구분한다.

## 변경 범위

- 헤더 기본·스크롤 배경색을 `#f6efe3`에서 `#efe5d4`로 변경한다.
- 헤더의 현재 투명도, 블러, 그림자, 상단 포인트 라인은 유지한다.
- 모바일 메뉴 본문과 하단 문의 영역의 배경도 `#efe5d4` 계열로 맞춘다.
- 푸터 배경을 `stone-900`에서 브랜드 다크 브라운 `#170c08`로 변경한다.
- 헤더·푸터의 텍스트, 버튼, 테두리, 링크 호버 색상은 변경하지 않는다.
- 헤더에 이미 적용 중인 높이, 로고 크기, 내비게이션 크기 변경은 보존한다.

## 구현 대상

- `src/widgets/header/ui/Header.tsx`
- `src/widgets/footer/ui/Footer.tsx`

## 검증

- 소스 검사로 헤더와 모바일 메뉴에 `#efe5d4`, 푸터에 `#170c08`이 적용되었는지 확인한다.
- 기존 헤더 배경 `#f6efe3`과 푸터 배경 `bg-stone-900`이 해당 컴포넌트의 배경에서 제거되었는지 확인한다.
- 기존 텍스트·버튼·테두리 색상 및 헤더 크기 관련 변경이 유지되는지 확인한다.
- ESLint와 프로덕션 빌드를 실행한다.
