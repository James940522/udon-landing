# ABOUT US 상·하단 에디토리얼 프레임 설계

## 목표

밝아진 `ABOUT US` 섹션의 본문과 카드는 유지하면서 상단과 하단에 편집형 장치를 추가해
섹션의 시작과 끝을 분명히 한다.

## 상단

- 섹션 최상단에 4px 적갈색 선을 추가한다.
- 콘텐츠 폭에 맞춘 인덱스 행을 상단 안쪽에 배치한다.
- 인덱스는 `01 / BRAND STORY`와 `TODAY UDON ARCHIVE`로 구성한다.
- 인덱스 양쪽에 작은 금색 원형 장식을 둔다.
- 장식은 콘텐츠와 겹치지 않도록 절대 위치와 높은 z-index를 사용한다.

## 배경 질감

- 기존 사진과 밝은 웜 오버레이 위에 매우 옅은 점무늬를 추가한다.
- 점무늬는 `rgba(143,53,40,0.16)`과 22px 간격을 사용한다.
- 본문 가독성을 해치지 않도록 낮은 투명도를 유지한다.

## 하단

- 섹션 하단에 1px 금색 구분선을 추가한다.
- 아래쪽 중앙에 아이보리색 타원형 종이 레이어를 절반만 보이도록 배치한다.
- 레이어는 다음 섹션으로 넘어가는 부드러운 곡선을 만들되 콘텐츠를 가리지 않는다.

## 비변경 범위

- 섹션 높이, 사진, 텍스트, 카드, 애니메이션은 변경하지 않는다.
- 다른 섹션은 수정하지 않는다.

## 검증

- 소스 계약 테스트로 인덱스 문구, 상단 선, 점무늬, 하단 선과 곡선 레이어를 확인한다.
- 기존 밝은 ABOUT US 계약을 함께 실행한다.
- ESLint와 프로덕션 빌드를 실행한다.
- 개발 서버는 실행하지 않는다.
