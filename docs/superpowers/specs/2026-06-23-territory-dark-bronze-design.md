# 상권 보장 다크 브론즈 디자인

## 목표

상권 보장 섹션의 밝은 종이 인상을 제거하고, 다크 초콜릿·브론즈·앤티크 골드로 구성한
묵직한 지도 분석실 분위기로 전환한다.

## 색상

- 배경: `#21160f`에서 `#35251b`로 이어지는 다크 초콜릿
- 제목: 웜 아이보리 `#f0dfc0`
- 본문: 샌드 베이지 `#c4ae91`
- 강조: 브론즈 `#c18442`
- 프레임: 앤티크 골드 `#a9824c`
- 비교 카드: 차콜 브라운 `#2b211a`, 적갈색 `#71422f`
- 안정성 패널: 구운 베이지 `#c5a77b`

## 배경 장식

- 두 겹의 얇은 골드 프레임을 섹션 가장자리에 배치한다.
- 낮은 투명도의 지도 좌표 격자를 전체 배경에 적용한다.
- 우상단과 좌하단에 추상적인 영업구역 경계선 SVG를 배치한다.
- 작은 좌표 레이블을 추가해 지도 분석실 인상을 강화한다.

## 유지 범위

- 문구, 지도 이미지, 비교 구조, 반응형 레이아웃과 애니메이션은 유지한다.
- 브라우저 시각 검증은 생략하고 계약 테스트, 린트, 포맷, 프로덕션 빌드로 검증한다.
