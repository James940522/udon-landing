# 블루오션 배경 이미지 노출 조정 설계

## 목표

웜 에디토리얼 톤과 콘텐츠 가독성은 유지하면서, 배경 이미지 위의 밝은 종이색 베일을
조금 투명하게 만들어 이미지의 질감과 형태가 더 보이게 한다.

## 확정 변경

- 전체 종이색 베일을 `bg-[#f7eddb]/68`에서 `bg-[#f7eddb]/48`로 낮춘다.
- 상단 아이보리 그라데이션을 `from-[#fff8eb]/82`에서 `from-[#fff8eb]/68`로 낮춘다.
- 중앙 베이지 그라데이션을 `via-[#e8d4b8]/72`에서 `via-[#e8d4b8]/56`으로 낮춘다.
- 하단 브라운 그라데이션을 `to-[#c99676]/62`에서 `to-[#c99676]/48`로 낮춘다.

## 비변경 범위

- 배경 이미지의 밝기와 채도는 유지한다.
- 문제 패널, 문제 카드, 혜택 카드, 인용구 카드와 최종 메시지의 배경 투명도는 변경하지 않는다.
- 텍스트 색상, 문구, 레이아웃, 반응형 구조와 애니메이션은 변경하지 않는다.

## 검증

- 기존 웜 에디토리얼 계약 테스트의 배경 베일 토큰을 새 값으로 갱신한다.
- 테스트가 기존 값을 기준으로 먼저 실패한 뒤 새 값 적용 후 통과하는지 확인한다.
- 집중 ESLint와 프로덕션 빌드를 실행한다.
