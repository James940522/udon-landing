# 창업 문의 에디토리얼 상담 데스크 장식 설계

## 목표

웜 페이퍼 배경, 짙은 브랜드 메시지 패널과 아이보리 문의 폼을 유지하면서 창업 문의 섹션에
에디토리얼 상담 데스크의 인상을 더한다.

장식은 문의 행동을 방해하지 않고 섹션의 시작, 두 패널의 관계와 빠른 응답 메시지를 명확히
보여주는 역할만 한다.

## 확정 디자인

### 섹션 인덱스와 프레임

- 콘텐츠 위에 `FRANCHISE INQUIRY · 01` 인덱스 라인을 추가한다.
- 인덱스 양옆에는 낮은 불투명도의 적갈색 선을 배치한다.
- 데스크톱에서는 섹션 왼쪽 위와 오른쪽 아래에 얇은 적갈색 L자 코너 프레임을 추가한다.
- 배경 오른쪽에는 `START YOUR STORE` 세로 문구를 매우 낮은 불투명도로 배치한다.
- 코너 프레임과 세로 문구는 콘텐츠보다 낮은 레이어에서 동작하고 클릭을 방해하지 않는다.

### 패널 연결

- 데스크톱 2열 레이아웃에서 메시지 패널과 폼 카드 사이에 짧은 점선 연결선과 화살표를
  배치한다.
- 연결선은 적갈색의 낮은 불투명도를 사용하고, 두 카드보다 뒤에 위치한다.
- 연결선은 실제 데이터 흐름을 의미하는 기능 요소가 아니라 상담에서 신청으로 이어지는
  시각적 안내 장식이다.

### 메시지 패널 도장

- 메시지 패널 오른쪽 위에 원형 `24H RESPONSE` 도장을 추가한다.
- 도장은 적갈색 테두리와 골드·베이지 텍스트를 사용하며 약간 회전시킨다.
- 도장 내부에는 `24H`를 크게, `RESPONSE`를 작은 대문자로 배치한다.
- 전화번호 CTA와 제목을 가리지 않도록 데스크톱에서는 오른쪽 위, 모바일에서는 문구 아래
  흐름 안에 배치한다.

### 폼 라벨

- 폼 카드 상단에 `APPLICATION FORM` 라벨을 추가한다.
- 라벨은 적갈색 작은 대문자와 짧은 라인으로 구성한다.
- 기존 폼 필드의 시작 위치를 조금 내려 라벨과 상단 바가 겹치지 않게 한다.

## 반응형과 접근성

- 모바일에서는 큰 코너 프레임, 세로 배경 문구와 패널 연결선을 숨긴다.
- 인덱스 라인, 도장과 폼 라벨은 모바일에서도 유지한다.
- 추가하는 인덱스, 프레임, 세로 문구, 연결선, 도장과 폼 라벨은 모두
  `aria-hidden="true"`로 처리한다.
- 클릭, 탭, 입력과 제출 동작을 방해하지 않도록 `pointer-events-none`을 적용한다.

## 비변경 범위

- 섹션 배경색, 두 카드의 색상과 폼 컨트롤 스타일은 변경하지 않는다.
- 문구, 전화번호, SMS 링크, 폼 상태와 이벤트 핸들러, API 제출 로직은 변경하지 않는다.
- 고객 리뷰 섹션과 푸터는 수정하지 않는다.
- 외부 패키지와 새 이미지 에셋은 추가하지 않는다.

## 검증

- 기존 문의 섹션 계약에 인덱스, 코너 프레임, 연결선, 도장, 폼 라벨과 모바일 숨김 토큰을
  추가한다.
- 모든 큰 장식이 `pointer-events-none`, `aria-hidden`과 데스크톱 전용 클래스를
  갖는지 확인한다.
- 기존 문의 섹션 색상·기능 계약을 유지한다.
- 계약 테스트를 RED에서 GREEN으로 확인하고 집중 ESLint, 현행 소스 계약, 전체 Node 테스트와
  프로덕션 빌드를 실행한다.
