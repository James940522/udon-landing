# 가맹절차 밝은 우드 배경 디자인

## 목표

가맹절차 섹션의 기존 어두운 나무 배경을 사용자가 제공한 밝은 우드 이미지로 교체하고, 섹션의 모든 텍스트·카드·번호 뱃지·CTA 색상을 사이트의 기존 브랜드 팔레트에 맞춘다.

## 선택한 방향

밝은 우드 이미지를 WebP로 최적화해 배경으로 사용한다. 이미지의 밝은 결을 가리지 않도록 오버레이는 아주 옅은 크림색으로 제한하고, 콘텐츠는 다른 밝은 섹션에서 사용하는 잉크 브라운·크림·라커 적갈색·골드 계열로 조정한다.

검토한 다른 방식은 기존 흰색 텍스트를 유지하기 위한 어두운 오버레이와 이미지 자체의 브랜드색 틴팅이었다. 두 방식 모두 새 이미지의 밝은 질감을 약화하므로 적용하지 않는다.

## 시각 디자인

- 배경 이미지는 `public/new-asset/background/startup-process-light-wood.webp`에 저장한다.
- 배경은 섹션 전체를 채우도록 `cover`, 중앙 정렬, 반복 없음으로 표시한다.
- 밝은 영역에서도 카드와 텍스트가 안정적으로 읽히도록 낮은 불투명도의 크림 오버레이를 둔다.
- 영문 라벨은 진한 브라운 배경과 골드 텍스트를 사용한다.
- 제목은 잉크 브라운, 소개 문구와 보조 문구는 중간 브라운을 사용한다.
- 기존 검정 반투명 소개 박스와 흰색 텍스트 그림자는 제거한다.
- 카드 표면은 반투명 크림, 테두리는 부드러운 브라운/골드, 그림자는 따뜻한 브라운 계열로 바꾼다.
- 카드 제목은 잉크 브라운, 설명은 중간 브라운을 사용한다.
- 번호 뱃지는 라커 적갈색 바탕과 크림색 숫자로 통일한다.
- 구분선은 라커 적갈색 또는 골드 계열로 조정한다.
- CTA는 진한 브라운 바탕, 골드 텍스트로 구성해 문의 섹션의 버튼 스타일과 연결한다.

## 동작과 범위

- 기존 6단계 데이터, 이미지, 애니메이션, 링크, 반응형 그리드는 유지한다.
- 변경 범위는 가맹절차 섹션과 새 WebP 배경 에셋으로 한정한다.
- 사용자가 작업 중인 메뉴 및 리뷰 섹션 파일은 수정하지 않는다.

## 검증

- 소스 계약 테스트에서 새 WebP 경로, 밝은 배경 설정, 브랜드 컬러 클래스가 적용되었는지 확인한다.
- ESLint와 프로덕션 빌드를 실행한다.
- 데스크톱과 모바일 레이아웃에서 배경 잘림, 텍스트 대비, 카드 가독성을 확인한다.
