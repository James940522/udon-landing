# 가맹절차 에디토리얼 장식 디자인

## 목표

밝은 우드 배경과 크림 카드 구성을 유지하면서, 가맹절차 섹션에 작은 에디토리얼 장식을 추가해 섹션의 고유한 인상과 6단계 흐름을 강화한다.

## 확정 디자인

- 헤더 위에 `6 STEPS TO OPEN` 문구를 추가하고 양옆에 얇은 골드 라인을 배치한다.
- 기존 `FRANCHISE PROCESS` 배지는 유지하되 헤더 장식과 함께 하나의 정보 위계로 정리한다.
- 각 카드 이미지 오른쪽 위에 작은 크림색 `STEP 01` 형식의 라벨을 추가한다.
- 데스크톱과 태블릿에서는 카드 뒤에 얇은 점선 연결 경로를 둔다.
  - 첫째 줄은 왼쪽에서 오른쪽으로 이어진다.
  - 오른쪽 끝에서 아래로 내려가 둘째 줄로 연결되는 ㄷ자 흐름을 만든다.
  - 카드 표면이 연결선을 덮어 카드 사이에서만 선이 보이게 한다.
- 섹션 왼쪽 위와 오른쪽 아래에 얇은 골드 L자 프레임을 배치한다.
- 모바일에서는 연결선과 큰 모서리 프레임을 숨기고 `STEP` 라벨만 유지한다.

## 스타일

- 장식선: `#c9a24d` 또는 `#a66732`의 낮은 불투명도.
- 장식 문구: 적갈색 `#8f3528`, 작은 대문자, 넓은 자간.
- STEP 라벨: 반투명 크림 바탕, 브라운 테두리, 잉크 브라운 텍스트.
- 모든 장식은 콘텐츠보다 낮은 시각적 강도를 유지하며 우드 결을 가리지 않는다.

## 동작과 범위

- 기존 배경 이미지, 6단계 데이터, 카드 이미지, 번호 뱃지, 애니메이션, CTA 링크는 변경하지 않는다.
- 새 이미지 에셋이나 외부 패키지는 추가하지 않는다.
- 변경 범위는 `StartupProcessSection`과 해당 소스 계약 테스트로 한정한다.
- 사용자가 작업 중인 메뉴 및 리뷰 관련 파일은 수정하지 않는다.

## 검증

- 소스 계약 테스트로 헤더 장식 문구, STEP 라벨, 데스크톱 전용 연결선, 모서리 프레임이 존재하는지 확인한다.
- 모바일 숨김 클래스가 연결선과 모서리 프레임에 적용됐는지 확인한다.
- 기존 배경·색상 계약 테스트를 유지한다.
- ESLint와 프로덕션 빌드를 실행한다.
