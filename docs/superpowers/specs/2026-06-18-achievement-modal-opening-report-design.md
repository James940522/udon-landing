# 오픈 현황 팝업 리포트형 디자인

## 목표

기존의 짙은 브라운 포스터 헤더, 동일 크기 매장 타일 격자, 큰 전화 배너 조합을 제거한다. 오픈 현황을 과장된 이벤트 광고가 아니라 실제 확장 기록을 정리한 “전국 오픈 리포트”처럼 보여준다.

## 선택한 방향

- 밝은 아이보리 종이를 기본 배경으로 사용한다.
- 상단은 얇은 적갈색 선과 편집형 타이포그래피로 구성한다.
- 전체 점포 수는 큰 숫자 하나로 강조하고, 브랜드명과 성과 문구는 보조 정보로 낮춘다.
- 매장은 카드 타일 대신 번호, 지역명, 오픈 상태가 한 줄에 정리된 리스트로 표시한다.
- 데스크톱에서는 두 열, 모바일에서는 한 열로 배치한다.
- 가맹 문의 전화는 큰 광고 배너 대신 리스트 하단의 간결한 안내 줄로 제공한다.
- 현재 월 오픈과 예정 점포는 작은 상태 배지로만 구분하고 반짝임 애니메이션은 제거한다.

## 반응형과 사용성

- 팝업은 화면 높이를 넘지 않도록 최대 높이를 지정하고 매장 목록 영역만 스크롤한다.
- 모바일에서도 점포 번호, 지역명, 상태가 한 줄에서 읽히도록 글자 크기와 간격을 유지한다.
- “오늘 하루 창 열지 않기”와 닫기 기능은 유지하되 리포트형 하단 바 스타일을 사용한다.
- 로딩 및 빈 상태도 동일한 밝은 종이 톤으로 표시한다.

## 성공 기준

- 기존 포스터형 헤더와 매장 타일 격자가 남지 않는다.
- 매장 목록이 모바일 한 열, 데스크톱 두 열의 리포트형 행 목록으로 보인다.
- 전체 점포 수, 오픈 상태, 전화 문의 정보가 명확히 읽힌다.
- 기존 모달의 숨김 및 닫기 동작이 유지된다.
